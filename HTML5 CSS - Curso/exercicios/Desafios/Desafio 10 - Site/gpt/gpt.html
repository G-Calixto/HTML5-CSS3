Minhas Habilidades

HTML5

HTML5

Linguagem de marcação usada para estruturar conteúdo na web.

Prós: Fácil de aprender, suportado em todos os navegadores, flexível.

Contras: Limitado sem CSS/JS, vulnerável a má estruturação.

CSS3

CSS3

Estiliza o HTML para tornar as páginas visualmente atraentes.

Prós: Responsivo, flexível, animações e transições.

Contras: Pode ser complexo em projetos grandes, dependente de compatibilidade.

JavaScript

JavaScript

Linguagem de programação que adiciona interatividade ao site.

Prós: Alta interatividade, popularidade, compatibilidade com o front-end.

Contras: Problemas de segurança, complexidade crescente em grandes projetos.
